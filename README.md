# FileShift Landing Page

Static landing page for the FileShift Android app, ready to publish with GitHub Pages.

## Local preview

Open `index.html` directly in a browser or run a static server from the repository root.

## Deploy with GitHub Pages

1. Push the repository to GitHub.
2. In the repository settings, open `Pages`.
3. Set the source to `Deploy from a branch`.
4. Select the `main` branch and the `/ (root)` folder.
5. Save the settings and wait for the site to publish.
